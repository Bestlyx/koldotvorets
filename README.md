# «Колдотворец» — лендинг

Откройте `index.html` в браузере. Сборка и сервер не требуются.

## Что подготовить перед публикацией

- Замените три текста-заглушки в блоке «Слово за игроками» настоящими отзывами.
- В блоке «От создателя» укажите имя автора вместо «Имя создателя».
- Ссылка «Купить PDF» ведёт в сообщения сообщества `vk.me/dndwithmeaning`; проверьте, что короткая ссылка открывает сообщения сообщества для посетителей.

Все иллюстрации находятся в `assets/` и уже подключены к странице.
